Lesson 3: Display, Div, and Span

In this lesson, I learned about how CSS controls how elements are displayed using the display property. Elements can be block-level or inline, which affects how they appear on a webpage. I also learned that div tags are used to group sections and are block-level, while span tags are used for styling small inline parts of text. Lastly, I learned that using display: block, inline, or none changes how elements are arranged or shown on the page.
