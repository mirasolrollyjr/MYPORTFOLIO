  1. Q2 L1
    2.  In this lesson I have learned how to use hyperlinks and how to create them. Its pretty simple really, just
  use the a tags and add an href in the opening tag. There are 3 interactive states hover, active, and visited. There are also 3 types of
  hyperlinks, they are absolute URL, relative URL, and named anchor.

  2. Q2 L2
    3.  In this lesson I have learned how to use Cascading Style Sheets (CSS) to style Web
  pages. CSS seperates content from design, making it easier to manage and update. Ther are three types of CSS - external, embedded, and inline
  each used for different purposes. I also learned about selectors, properties, and values, which define how elements look.
  Over all, CSS helps make web pages more oranized and visually appealing.

  3. Q2 L3
    4. I also learned how to use the div and span tags: the div
   tag is used to group large sections of a page, while the span tag is used
    to style small portions of text. Understanding these concepts helps
     create cleaner and more organized webpage designs.
  4. Q2 L4
    5. In this lesson, I learned about CSS rules, which are made up of selectors and style 
declarations that define how HTML elements appear on a webpage.
I also explored how to use classes to apply styles to specific elements, making web
design more organized and flexible. Classes can be reused across different tags and
 combined with element names for more precise styling. Lastly, I learned how to group
 selectors to apply the same style to multiple elements efficiently.
  5. Q2 L5
    6. In this lesson, I learned how to use an external style sheet to apply 
 CSS to multiple web pages at once. It makes the design more consistent and easier 
to update because all styles are stored in a single CSS file linked to the HTML.
  6. Q2 L6
    7. In this lesson, I learned how the CSS position property controls where elements
appear on a webpage. I explored different positioning types like 
static, relative, absolute, and fixed, which help organize the layout more precisely.
