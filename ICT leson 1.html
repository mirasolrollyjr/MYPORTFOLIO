CSS stands for Cascading Style Sheets. It defines how HTML elements are displayed, making it possible to create layouts, colors, and fonts that improve the look and feel of a website. CSS allows one stylesheet to control the design of multiple web pages, saving time and effort. It also separates the structure (HTML) from style, improving consistency, accessibility, and page performance.

A webpage is made up of three layers:

  • Content Layer â€“ defines the text, images, and other elements using HTML.
  • Presentation Layer â€“ defines how content is displayed using CSS.
  • Behavior Layer â€“ defines how the page reacts to user interaction, often using JavaScript.
