CSS (Cascading Style Sheets) is used to describe how HTML elements are displayed on a webpage. It separates content (HTML) from design (CSS), making pages easier to manage and more visually consistent.

üß© Parts of CSS

  • Selector ‚Äì the HTML element to style.
  • Property ‚Äì the characteristic to change (e.g. color, font-size).
  • Value ‚Äì the setting for the property (e.g. red, 14px).
  • Declaration ‚Äì one property-value pair.
  • Declaration Block ‚Äì a group of declarations inside braces { }.
Example:

üìò How Style Sheets Work

  • CSS tells the browser how to display HTML elements.
  • Content is in HTML, while appearance is in CSS.
  • Use <style type="text/css"> to define CSS in HTML.
  • Older browsers ignore CSS inside comments <!-- ... -->.
